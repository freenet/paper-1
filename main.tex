\begin{abstract}
\noindent
Freenet is a decentralized application platform built on a peer-to-peer network
of programmable, eventually-consistent key-value cells called \emph{contracts}.
Each contract is a WebAssembly module that defines its own
\emph{commutative monoid} on application state, together with a generic
summary/delta synchronization protocol that lets peers efficiently reconcile
divergent replicas without global coordination. Contracts are located on a
one-dimensional ring by content hash and reached using \emph{adaptive small-world
routing}: peers learn a distance-indexed performance model for each neighbor and
forward each request to the neighbor expected to deliver the answer fastest, not
necessarily the neighbor topologically closest to the destination. The
small-world topology itself is constructed by a single rule --- \emph{accept a
connection only at the routing terminus} --- which causes local clustering and
short paths to emerge without explicit long-link construction. Alongside
public contract state, Freenet provides \emph{delegates}: actor-style
WebAssembly agents that hold private secrets on a user's device and respond
only to authenticated messages. The result is a platform that supports
real-time, interactive decentralized applications --- such as group chat,
collaborative documents, and reputation systems --- with eventual consistency
defined by the application itself rather than by the network.
\end{abstract}
